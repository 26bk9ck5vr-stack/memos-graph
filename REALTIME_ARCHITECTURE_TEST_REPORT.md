## 问题分析

### 核心问题：召回失败

**现象**:
- ✅ 数据已写入数据库
- ✅ 向量已生成并保存
- ❌ 召回 API 返回旧数据

**可能原因**:
1. FTS (全文搜索) 索引未更新
2. 向量相似度查询未包含新数据
3. 召回逻辑有过滤条件排除新数据
4. tsvector 字段未正确生成

**验证方法**:
```sql
-- 检查 FTS 索引
SELECT id, tsvector FROM chunks 
WHERE id IN (4529, 4673);

-- 检查向量
SELECT chunk_id, embedding FROM chunk_vectors 
WHERE chunk_id IN (4529, 4673);
```
